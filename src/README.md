 1. Консольна система управління університетом (Java CLI)

Це фінальний проєкт, розроблений мовою Java. Програма імітує базову систему управління університетом та працює через інтерфейс командного рядка (CLI).

 2. Функціонал програми

Проєкт реалізує повноцінні CRUD-операції та бізнес-логіку для основних сутностей університету:

Студенти: Додавання, перегляд, зміна статусу (ACTIVE, ON_LEAVE, EXPELLED, GRADUATED), сортування за ПІБ (Bubble Sort).

Викладачі: Додавання та перегляд з використанням посад (ASSISTANT, LECTURER, PROFESSOR).

Курси: Створення навчальних курсів, призначення викладачів та кількості кредитів.

Зарахування (Enrollments): Запис студентів на курси, виставлення оцінок (A, B, C, D, F, NA), проведення оплат.

Звіти: Виведення списку боржників (неоплачені курси), генерація транскрипту студента з автоматичним розрахунком середнього балу (GPA).

 3. Технології та принципи ООП

У проєкті використано:

Java 17+ (Core Java).

Принципи ООП: Інкапсуляція, Наслідування (базовий клас Person), Поліморфізм.

Інтерфейси: Реалізовано інтерфейс Payable для роботи з оплатами.

Enums: Використано для статусів, оцінок та посад.

Collections: Використання ArrayList для зберігання даних у пам'яті програми.

Обробка винятків: try-catch та генерування IllegalArgumentException при некоректному вводі (наприклад, невірний email чи рік навчання).

 4. Структура проєкту

src/university/Main.java — головний клас із меню навігації.

src/university/entities/ — класи-сутності (Student, Teacher, Course, Enrollment).

src/university/enums/ — перелічення.

src/university/interfaces/ — інтерфейси.

src/university/services/ — бізнес-логіка та зберігання масивів даних.

src/university/util/ — утиліти (розрахунок GPA).

 5. Як запустити проєкт

Переконайтеся, що на вашому комп'ютері встановлено Java JDK (версії 17 або вище).

Склонуйте цей репозиторій:

git clone [ТУТ_БУДЕ_ТВОЄ_ПОСИЛАННЯ_НА_GITHUB]


Відкрийте проєкт у вашій улюбленій IDE (VS Code, IntelliJ IDEA, Eclipse).

Запустіть файл src/university/Main.java.

Керуйте програмою, вводячи цифри меню в консоль.

Проєкт виконано в рамках навчального курсу.
